Hébergement de développement via environnement local (serveur Apache / PHP / MySQL).

3. Objet du site :

Kiyomi est un site web permettant de commander des sushis et autres spécialités japonaises en ligne.

Le site propose la consultation d’un menu, la création d’un compte utilisateur, ainsi que la possibilité de passer des commandes.

Kiyomi est un projet réalisé dans le cadre d’un travail universitaire et n’a pas de vocation commerciale réelle.

4. Propriété intellectuelle :

L’ensemble des contenus présents sur le site Kiyomi (textes, visuels, logo, interface, code source) sont protégés par le droit d’auteur.

Toute reproduction, représentation ou utilisation sans autorisation est interdite, sauf dans un cadre strictement pédagogique.

Les images et visuels utilisés le sont à des fins éducatives et démonstratives.

5. Données personnelles :

Les données personnelles collectées sur le site Kiyomi (nom, prénom, email, mot de passe) sont utilisées uniquement dans le cadre du fonctionnement du site.

Aucune donnée n’est cédée à des tiers. Les informations sont stockées de manière sécurisée.

Conformément au RGPD, chaque utilisateur peut demander la modification ou la suppression de ses données.

Kiyomi est un site développé dans un cadre pédagogique dans le cadre du BUT MMI.
